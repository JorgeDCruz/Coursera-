## Testing with Jasmine
**Set Up:** 

*Global Instalation:*
``` bash
npm install –g jasmine
```

*Development Instalation:*
``` bash
npm install –-save-dev jasmine
```

*Module initialization:*
``` bash
c:\>node node_modules/Jasmine/bin/Jasmine init 
```

*Script initialization:* 

In **package.json** add: 
``` json
"scripts":{
    "test": "jasmine"
}
```
